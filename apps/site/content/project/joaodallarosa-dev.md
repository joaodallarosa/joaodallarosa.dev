Mock content for layout verification — not a real project write-up yet.

The site itself, built to double as a working lab: a small design-system package of plain Web Components, content authored through Nuxt Content and editable live via Nuxt Studio, and locale-aware routing across English, French, and Brazilian Portuguese.

Most pages are statically generated at build time; Nuxt Studio's editor keeps running on a live server so content changes can go out without a full redeploy cycle in mind for every typo fix.
