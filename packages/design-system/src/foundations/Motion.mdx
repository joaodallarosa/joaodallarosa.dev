# Motion

Two registers. **Mechanical** (`--motion-easing-mechanical`, a snappy `cubic-bezier(0.2, 0, 0, 1)`)
is the default for everyday micro-interactions — buttons, tags, borders. **Fluid**
(`--motion-easing-fluid`) is reserved for overlay/depth moments only (glass panels). All durations
collapse under `prefers-reduced-motion: reduce` (see the media query in `tokens.css`) — toggle it
in your OS/browser settings to verify rather than trusting the token value alone.
