# Typography

Two families: `--font-family-serif` (Fraunces, editorial voice — headlines, body prose) and
`--font-family-mono` (JetBrains Mono, structural/UI voice — labels, badges, Developer Mode).
Font self-hosting is still deferred (currently falling back to Georgia/monospace); see
`PROJECT.md` Phase 2.

export const Sample = ({ varName, family = '--font-family-serif', weight }) => (
  <p
    style={{
      fontFamily: `var(${family})`,
      fontSize: `var(${varName})`,
      fontWeight: weight ? `var(${weight})` : undefined,
      color: 'var(--color-text)',
      margin: '0 0 0.75rem',
    }}
  >
    <code style={{ fontSize: '0.7em', color: 'var(--color-text-muted)', marginRight: '0.75em' }}>
      {varName}
    </code>
    The quick brown fox
  </p>
)

## Display / hero scale — fluid, shell-only

`clamp()`-based, deliberately oversized for bleed-friendly shell moments (homepage hero, section
dividers) per the "shell bold, reading calm" split in `docs/design-layout-references.md` — resize
the preview frame to see them scale with the viewport. Never used in long-form reading body text.
