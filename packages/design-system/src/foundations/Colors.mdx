# Colors

Warm-tinted neutrals (never pure black/white) plus a single confident accent — phosphor amber.
Kind badges use a separate, desaturated palette so content-kind color never gets confused with
the accent or with status. Full rationale in `docs/design-prompt.md`. Dark is the default scheme;
`[data-color-scheme="light"]` overrides the surface/text/accent roles below — use the toolbar's
color-scheme control above to compare.
