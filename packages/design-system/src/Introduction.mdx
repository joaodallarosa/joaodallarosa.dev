# joaodallarosa.dev — design system

Lit-based Web Components, consumed live by [joaodallarosa.dev](https://joaodallarosa.dev) — not
mockups. Full rationale for every decision lives in `docs/design-prompt.md` (tokens) and
`PROJECT.md` §3–4 (architecture, Developer Mode).

- **Foundations** — color, typography, spacing, motion tokens, with live swatches.
- **Components** — `ds-button`, `ds-tag`, `ds-badge`, `ds-card`, each with real Shadow DOM,
  controls generated from the component's own TypeScript source (via
  `@custom-elements-manifest/analyzer`), and the same Developer Mode x-ray skin shipped on the
  live site — toggle it from the toolbar above.

Leaf/true components get real Shadow DOM; layout-ish/structural site chrome doesn't (see
`PROJECT.md` §3 for why) — this reference only covers the former.
